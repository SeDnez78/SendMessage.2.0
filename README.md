# Enviar Mensaje a WhatsApp desde Web

## Descripción
Aplicación web que permite enviar mensajes directamente a WhatsApp 
desde el navegador. Desarrollada con HTML, CSS y JavaScript puro.

##  Demo en vivo
[Visitar la web aquí](https://SeDnez78.github.io/EnviarMensaje/)

##  Tecnologías
- HTML5
- CSS3
- JavaScript (Vanilla)

##  Cómo usar
1. Abre el [index.html](./index.html) en tu navegador.
2. Escribe tu mensaje y haz clic en enviar.
